Developer Guide

Overview
********

This section was designed to lead new developers in understanding the
Kytos project. First, we recommend the reading of the Kytos architecture in
:doc:`Admin Guide </administrator/architecture>`. There you will learn
how to install and configure Kytos and the NApps, as well as understand the
develop environment. Then, give a try to
the `tutorials <http://tutorials.kytos.io/>`__ made by Kytos Team.
